# Requirements and Venv Setup

This file contains exact setup commands and installed requirements for this project.

## 1) Create and activate venv (WSL)

```bash
cd /mnt/f/Downloads/6th_Sem_EL/CD_Lab
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
```

## 2) System tools required (WSL)

```bash
sudo apt update
sudo apt install -y python3 python3-venv python3-pip clang llvm clang-tidy cppcheck make
```

## 3) Python requirements install

Use the exported requirements snapshot file:

```bash
source .venv/bin/activate
pip install -r requirements.txt
```
